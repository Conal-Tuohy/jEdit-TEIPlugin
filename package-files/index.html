Introduction

This plugin provides tools for text encoding using an XML vocabulary called Text Encoding for Interchange (TEI).

For more information about TEI, visit the website of the Text Encoding Initiative Consortium .

Development of the initial version of the plugin was funded by DARIAH-EU in 2018/2019.

Toolbar buttons and keyboard shortcuts

The TEI toolbar contains the following buttons:

  • Click the Update TEI Package button or type C+u to manually trigger the download of a package of TEI-related files from the TEI Consortium's website.

    To specify a different location to download the package from, see the options dialog.

  • Click the New TEI file button, or type C+n , to open a hierarchical menu of TEI template files, provided by the TEI package. Selecti one of the templates to create a new document based on that template.

  • Click the Parse Buffer button, or type CS+w , to parse the current buffer. Any schema invalidities will be reported in the Error List panel.

  • If the cursor is positioned on either an opening or closing tag of an XML element, you can click the Go to Matching Tag or Bracket button,r type CS+m , to move the cursor to the other matching tag; i.e. from the opening to the closing tag, or from the closing tag to the opening tag.

  • Click the Selection between tags button, or type CS+t to expand the selected portion of the document to include the content of the parent XML element.

  • Click the Split Tag button, or type CS+d , to split the parent XML element in two, at the location of the cursor.

  • Click the Index XML button, or type CS+p to indent the XML document.

These operations are also listed in the menu which appears for TEI under the Plugins heading.

Options

There are two options which can be set using the pane for TEI in the Plugin Shortcuts section of the general options dialog:

  • The Auto-update TEI package checkbox causes the TEI Plugin to check for updates to the TEI package and download them automatically.

  • The TEI package update URL field specifies the location of a metadata resource which lists the avaiable versions of the TEI package. When the TEI Consortium updates this package metadata resource to add a reference to a new package, the TEI Plugin will download and install the new package.

    This URL can be changed, which can be useful if you are building a new version of the TEI package and want to test it.
